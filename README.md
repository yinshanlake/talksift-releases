# TalkSift

**Voice to polished text.** Speak naturally, get clean, structured output.

TalkSift is a desktop overlay app that transcribes your speech and uses AI to extract, summarize, or translate it — all with a single click.

## Download

👉 **[Latest Release](https://github.com/yinshanlake/talksift-releases/releases/latest)**

Download the ZIP, extract, and run `TalkSift.exe`.

## Features

- 🎙️ One-click voice recording with real-time duration display
- ✨ Multiple AI processing modes: Extract, Summarize, Translate, Custom
- 📋 Auto-copy results to clipboard
- 🌐 Supports Chinese, English, and mixed-language input
- 🖥️ Floating overlay — always accessible, never in the way
- ⚡ Local Whisper STT (no audio leaves your machine)

## System Requirements

- Windows 10/11
- Microsoft Edge WebView2 Runtime (usually pre-installed)
- Microphone
